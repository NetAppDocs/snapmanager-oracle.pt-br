= Reiniciando o servidor host SnapManager UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode reiniciar o servidor em um host UNIX usando a CLI.

. Para reiniciar o servidor, digite o seguinte comando:
`smo_server restart`
