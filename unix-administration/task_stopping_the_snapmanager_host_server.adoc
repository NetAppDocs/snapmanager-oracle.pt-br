= Parando o servidor host do SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Quando terminar de usar o SnapManager, talvez você queira parar o servidor.

. Para parar o servidor, digite o seguinte comando, como usuário root:
`smo_server stop`
